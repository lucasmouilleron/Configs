Install - Ubuntu
-----------------
- From root or sudo user:`usermod -aG sudo username`
- From user `username`: `sudo ls`
- From user `username`: `cd $HOME;sh <(wget -o /dev/null -qO- https://raw.githubusercontent.com/lucasmouilleron/Survival/master/ubuntu.sh)`
